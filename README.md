# Writer dashboard project (Vue)

## What is it ?

This project is a meant to be an inspiration dashboard for writers, illustrators, creatives as a whole. It generates quotes, a few images and words to find some inspiration in different ways. 

This is still a work in progress, no real design at the moment.

I used <a href="https://www.wordsapi.com/">WordApi</a> for the random words generator, <a href="https://forum.freecodecamp.org/t/free-api-inspirational-quotes-json-with-code-examples/311373">this API</a> for the quotes and <a href="https://picsum.photos/">Lorem Picsum</a> for the images. 

### Français
Ce projet est pensé comme un tableau de bord pour écrivains, illustrateurs, créatifs en général. Il génère des citations, quelques images et mots aléatoires pour trouver un peu d'inspiration de diverses manières.

C'est encore en cours de conception, pas de design pour l'instant.

J'ai utilisé <a href="https://www.wordsapi.com/">WordApi</a> pour les mots générés aléatoirement, <a href="https://forum.freecodecamp.org/t/free-api-inspirational-quotes-json-with-code-examples/311373">cette API</a> pour les citations et<a href="https://picsum.photos/">Lorem Picsum</a> pour les images.
